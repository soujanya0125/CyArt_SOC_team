# Task 1: Alert Management

## Objective
Learn to manage and categorize security alerts in Wazuh dashboard.

## Prerequisites
- Wazuh instance running
- Access to Wazuh dashboard
- Sample alerts generated

## Steps

### 1. Access Wazuh Dashboard
- Navigate to Wazuh web interface (typically http://localhost)
- Log in with admin credentials
- Access the Dashboard view

### 2. Alert Classification
Classify the following alerts:
- [ ] Malware detection
- [ ] Failed login attempts
- [ ] Unusual network traffic
- [ ] Process execution alerts
- [ ] File integrity monitoring alerts

### 3. Priority Assignment
For each alert, assign appropriate priority:
- **P1 (Critical)**: Immediate threat indicators
- **P2 (High)**: Significant security concerns
- **P3 (Medium)**: Potential issues requiring investigation
- **P4 (Low)**: Informational alerts

### 4. Dashboard Customization
- Create custom dashboard views
- Add alert filtering
- Set up alert rules
- Configure automated responses
